# Lab 🧪
Это репозиторий с разборами механик, примерами кода и небольшими учебными проектами на Unity.

Я создаю эти исходники как дополнение к своим видео и девлогам. Основная цель - показать рабочие решения без лишнего усложнения, которые можно быстро выдрать и внедрить к себе.

📦 Что внутри
Репозиторий будет постоянно пополняться новыми проектами. Сейчас доступно:

Donut Clicker - база для кликера. Прогрессия, визуальный отклик (Juice), партиклы, звуки и простая анимация.

Скоро здесь появится что-то новое...

🛠 Стек
В проектах я стараюсь придерживаться современных подходов, которые использую в своей основной разработке (например, в игре Deckart):

Unity (актуальные версии)

VContainer - для внедрения зависимостей (DI)

DOTween - для сочных анимаций

🚀 Как пользоваться
Клонируешь репозиторий.

Открываешь нужную папку проекта в Unity.

Разбираешь код, тыкаешь настройки и используешь в своих проектах.
